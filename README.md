## About

Computer Engineering student from India with a focus on building AI-powered systems, Android apps, and full-stack web projects. I care more about shipping real things than collecting tutorials.

Currently working on **DataVaani** — an AI-based environmental monitoring and AQI forecasting system built for **Bharatiya Antariksh Hackathon 2025**, integrating satellite data, a Prophet ML model, and an interactive frontend.

Outside of that: building Android tools, experimenting with local LLMs, and pushing toward custom AI systems with full personality and UI control.
